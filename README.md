# Справочник источников

Веб-приложение для просмотра и фильтрации списка источников для анализа трендов. Данные загружаются из Google Таблицы или из CSV-файла.

## Как открыть

- **Репозиторий (код):** [github.com/MatveevAE-dev/source_research](https://github.com/MatveevAE-dev/source_research)
- **Сайт (справочник):** после включения GitHub Pages откройте в браузере:  
  **https://matveevaedev-dev.github.io/source_research/**
- **Локально:** запустите в этой папке `python -m http.server 8080` или `npx serve .` и откройте в браузере указанный адрес.

Если сайт по ссылке не открывается — включите GitHub Pages: в репозитории **Settings → Pages** → Source: **Deploy from a branch** → Branch: **main**, папка **/ (root)** → **Save**. Подождите 1–2 минуты.

Подробная инструкция — в файле [ИНСТРУКЦИЯ.md](ИНСТРУКЦИЯ.md).
