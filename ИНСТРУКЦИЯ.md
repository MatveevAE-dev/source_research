# Справочник источников — инструкция

## Что это

Справочник источников — это веб-страница, на которой отображается список источников для анализа трендов. Данные загружаются из вашей Google Таблицы или из загруженного CSV-файла. Список сгруппирован по **направлениям** и **категориям**, по нему можно искать и фильтровать.

---

## Как пользоваться программой

### Открытие и загрузка данных

1. **Запустите сайт** (см. раздел «Как запустить программу с другого компьютера» ниже).
2. При первом открытии страница попытается **загрузить данные из Google Таблицы** по ссылке, указанной в настройках. Если загрузка не удалась (например, из-за ограничений доступа в браузере), появится сообщение с просьбой загрузить файл.
3. **Загрузить свой CSV:** нажмите кнопку **«Загрузить свой CSV»** в правом верхнем углу и выберите на компьютере файл в формате CSV с колонками: *Направление*, *Наименование источника*, *Ссылка*, *Комментарии*, *Категория*, *Тип источника*, *Страна*, *Список* (как в экспорте из Google Таблицы).

### Поиск

- В поле **«Поиск по названию или описанию…»** введите часть названия источника или текст из комментариев.
- Список обновляется сразу: показываются только те источники, в названии или описании которых есть введённый текст.

### Фильтры (направление, категория, страна)

- Под строкой поиска расположены три фильтра: **Направление**, **Категория**, **Страна**.
- Нажмите на кнопку фильтра (например, «Все направления») — откроется выпадающий список с чекбоксами.
- **Выберите один или несколько вариантов** — отмеченные пункты применяются как фильтр (показываются источники, подходящие хотя бы под один из выбранных вариантов в каждом включённом фильтре).
- На кнопке отображается: «Все …», один выбранный пункт или «Выбрано: N», если выбрано несколько.
- **Сбросить фильтр:** в открытом списке нажмите **«Сбросить»** вверху панели — все галочки в этом фильтре снимутся, снова будут показаны все варианты по этому полю.
- Чтобы закрыть панель без сброса, нажмите вне её или откройте другой фильтр.

### Просмотр источников

- Источники сгруппированы сначала по **направлению**, внутри — по **категории**.
- У каждого источника отображаются: **название**, **страна**, **комментарий** и ссылка **«Открыть источник»**.
- Нажмите **«Открыть источник»**, чтобы перейти на сайт/страницу источника в новой вкладке.

### Инструкция по категориям

- Кнопка **«Инструкция по категориям»** в шапке страницы открывает окно с подсказками: что означает каждая категория и когда её лучше использовать.
- Окно закрывается по кнопке **«×»**, клику по затемнённой области или клавише **Escape**.

## Как запустить программу с другого компьютера

Сайт состоит из обычных файлов (HTML, CSS, JS) и не требует установки. Чтобы открыть его на другом компьютере, нужно **скопировать папку проекта** на этот компьютер и **запустить локальный веб-сервер** в этой папке (иначе загрузка данных из файлов/таблицы может не работать).

### Вариант 1: Копирование папки и запуск на другом ПК

1. **Скопируйте всю папку проекта** (например, `source_research`) на другой компьютер (флешка, облако, архив по почте и т.п.). Внутри должны быть как минимум:
   - `index.html`
   - папка `css` с файлом `style.css`
   - папка `js` с файлом `app.js`
2. На том компьютере **откройте терминал (командную строку)** в этой папке.
3. Запустите один из вариантов сервера:

   **Если установлен Node.js:**
   ```bash
   npx serve .
   ```
   В браузере откройте адрес, который появится в терминале (обычно `http://localhost:3000`).

   **Если установлен Python 3:**
   ```bash
   python -m http.server 8080
   ```
   В браузере откройте: `http://localhost:8080`

4. Страница «Справочник источников» откроется в браузере; данные подгрузятся из Google Таблицы или после нажатия «Загрузить свой CSV».

### Вариант 2: Запуск без сервера (только загрузка своего CSV)

- Можно просто **открыть файл `index.html`** двойным щелчком (он откроется в браузере).
- Загрузка из Google Таблицы при открытии через `file://` часто не срабатывает. В этом случае нажмите **«Загрузить свой CSV»** и выберите файл реестра источников — справочник будет работать с загруженным файлом.

### Вариант 3: Размещение в интернете (доступ любому из интернета)

Чтобы сайт открывался по ссылке у **любого человека**, из любой сети, не нужно держать включённым свой компьютер. Нужно выложить файлы на **бесплатный хостинг**.

#### Способ A: Netlify Drop (самый простой)

1. Зайдите на [app.netlify.com/drop](https://app.netlify.com/drop).
2. Создайте на компьютере **папку** с тремя элементами:
   - файл `index.html` (из корня проекта);
   - папка `css` с файлом `style.css`;
   - папка `js` с файлом `app.js`.
3. **Перетащите эту папку** в окно Netlify Drop (или нажмите «Add your site» и выберите папку).
4. Netlify выдаст постоянную ссылку вида `https://случайное-имя.netlify.app`. Эту ссылку можно отправлять кому угодно — сайт будет доступен из интернета.
5. Данные по-прежнему подгружаются из вашей Google Таблицы (если у таблицы открыт доступ по ссылке).

#### Способ B: GitHub Pages

1. Зарегистрируйтесь на [GitHub](https://github.com), создайте новый репозиторий.
2. Загрузите в репозиторий файлы: `index.html`, папки `css` и `js` с содержимым.
3. В настройках репозитория: **Settings → Pages** → в разделе Source выберите ветку (например, `main`) и папку `/(root)` → Save.
4. Через минуту сайт будет доступен по адресу `https://ваш-логин.github.io/имя-репозитория/`.

#### Способ C: ngrok (временная ссылка с вашего ПК)

Если нужно быстро дать доступ **без регистрации на хостинге**, но компьютер должен быть включён:

1. Установите [ngrok](https://ngrok.com/download).
2. Запустите на своём компьютере сервер: `python -m http.server 8080`.
3. В другом окне терминала выполните: `ngrok http 8080`.
4. В выводе ngrok появится ссылка вида `https://xxxx.ngrok.io` — её можно отправить. Пока ваш ПК включён и работают сервер и ngrok, по этой ссылке откроется ваш сайт.

Минусы: ссылка меняется на бесплатном аккаунте при каждом запуске; ваш компьютер должен быть включён.

## Требования

- Современный браузер (Chrome, Firefox, Edge, Safari).
- Для запуска через `npx serve` — установленный [Node.js](https://nodejs.org/).
- Для запуска через Python — установленный [Python 3](https://www.python.org/).
- Для загрузки из Google Таблицы таблица должна быть доступна по ссылке («Просматривать могут все, у кого есть ссылка» или шире).

## Обновление данных из Google Таблицы

Если вы правите данные в Google Таблице, изменения появятся на сайте **после обновления страницы** в браузере (клавиша F5 или кнопка «Обновить»). Вручную ничего в коде менять не нужно.
